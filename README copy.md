# Book Search Engine

## Description

A Google Books API search engine refactored from a RESTful API to a GraphQL API built with the Apollo Server. Built using the MERN Stack.

## Technologies Used

- Node.js
- React
- MongoDB
- JSONWebToken
- Express.js
- GraphQL
